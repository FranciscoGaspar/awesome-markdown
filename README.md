## Contributing

Please feel free to [open an issue](https://github.com/yourusername/awesome-ai-scraper/issues) or [submit a pull request](https://github.com/yourusername/awesome-ai-scraper/pulls) to add more resources to this list.

### Guidelines

- **Quality over quantity**: Only include high-quality, well-maintained resources
- **Descriptive links**: Use descriptive link text that explains what the resource is
- **Categorization**: Place resources in the most appropriate category
- **No duplicates**: Check if a resource is already listed before adding
- **Active projects**: Prefer actively maintained projects over abandoned ones
