### AI-Enhanced Development

- [GitHub MCP Server](https://github.com/github/github-mcp-server?tab=readme-ov-file) - GitHub MCP Server connects AI tools to GitHub, enabling natural language interactions for repository management, issue automation, CI/CD insights, and code analysis.
- [MCP Atlassian](https://github.com/sooperset/mcp-atlassian) - MCP Atlassian is a server integration for Atlassian tools, supporting Confluence and Jira. It enables AI-powered features like automatic Jira updates and smart Confluence searches.
