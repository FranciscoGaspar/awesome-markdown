## AI-Enhanced Development

### AI-Enhanced Development

- [GitHub MCP Server](https://github.com/github/github-mcp-server?tab=readme-ov-file) - GitHub MCP Server connects AI tools to GitHub, enabling natural language interactions for repository management, issue automation, CI/CD insights, and code analysis.
- [MCP Atlassian](https://github.com/sooperset/mcp-atlassian) - MCP Atlassian is a server for integrating AI tools with Atlassian products like Confluence and Jira, supporting both cloud and server/data center deployments. It enables features like automatic Jira updates and AI-powered Confluence search.
