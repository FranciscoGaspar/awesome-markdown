# Awesome AI Resources

A curated list of awesome AI resources, tools, frameworks, and libraries.

## AI-Enhanced Development

### AI-Enhanced Development

- [Bitbucket MCP](https://github.com/MatanYemini/bitbucket-mcp) - Bitbucket MCP is a Model Context Protocol server enabling AI assistants to interact with Bitbucket repositories and pull requests without data deletion risks, ensuring secure code analysis.
- [GitHub MCP Server](https://github.com/github/github-mcp-server?tab=readme-ov-file) - GitHub MCP Server connects AI tools to GitHub, enabling natural language interactions for repository management, issue automation, CI/CD insights, and code analysis.
- [MCP Atlassian](https://github.com/sooperset/mcp-atlassian) - MCP Atlassian is a server for integrating AI tools with Atlassian products like Confluence and Jira, supporting both cloud and server deployments for enhanced automation and search capabilities.

## AI Governance

### AI Governance

- [AI Gateway](https://xgeeks.com/ai-gateway) - The AI Gateway by xgeeks is designed to control and safeguard AI usage within organizations, ensuring secure governance and alignment of tools for accelerated value.
